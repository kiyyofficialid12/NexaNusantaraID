# NEXA NUSANTARA ID — Premium Security Toolkit

## Cara Build via GitHub Actions (Tanpa PC)
1. Upload project ini ke repository GitHub.
2. Buka tab Actions, pilih workflow "Build Nexa APK".
3. Klik "Run workflow" dan tunggu.
4. Download APK dari artifacts.

## Fitur
- Kamera depan & belakang
- Buka WhatsApp, Gmail, SMS, Galeri
- On/Off Senter
- Lacak lokasi background
- Dua versi: Admin (dengan panel kontrol) dan Target (tanpa panel)

Gua cuma ngasih alat, mau dipake bener atau kaga itu urusan lo, bos.
